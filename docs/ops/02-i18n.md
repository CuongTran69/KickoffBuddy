## Mục đích

Định nghĩa chiến lược i18n cho Kickoff Buddy. App hỗ trợ tiếng Việt (default) và tiếng Anh. Tiếng Việt là ngôn ngữ chính cho thị trường mục tiêu.

## Phạm vi

### In scope

- ARB file structure
- String key naming convention
- Date/number formatting
- Language switching

### Out of scope

- RTL languages
- Localization cho thị trường ngoài VN/EN

---

## Ngôn ngữ hỗ trợ

| Locale | Tên | Default |
|---|---|---|
| `vi` | Tiếng Việt | Có |
| `en` | English | Không |

---

## ARB File Structure

```
lib/l10n/
├── app_vi.arb    ← Tiếng Việt (default)
└── app_en.arb    ← English
```

## String Key Naming Convention

Format: `<screen>_<element>_<variant>`

| Ví dụ | Giải thích |
|---|---|
| `onboarding_welcome_title` | Screen: onboarding, element: welcome, variant: title |
| `match_add_magic` | Screen: match, element: add, variant: magic |
| `reminder_default_1day` | Screen: reminder, element: default, variant: 1day |
| `error_network_generic` | Screen: error, element: network, variant: generic |

## Date/Number Formatting

Dùng `intl` package với locale:

```dart
// Ngày tháng
DateFormat('dd/MM/yyyy', 'vi').format(date)  // "12/06/2026"
DateFormat('EEEE, dd/MM', 'vi').format(date)  // "Thứ Sáu, 12/06"

// Giờ
DateFormat('HH:mm', 'vi').format(time)  // "02:00"
DateFormat('h:mm a', 'en').format(time)  // "2:00 AM"

// Countdown
// Tự implement: "còn 2 ngày 3 giờ" hoặc "còn 45 phút"
```

## Language Switching

- Default: theo ngôn ngữ thiết bị (nếu là tiếng Việt → vi, còn lại → en)
- User có thể override trong Settings
- Lưu trong `UserPreference.language`

---

## Edge cases

- Thiết bị dùng ngôn ngữ không hỗ trợ: fallback về tiếng Anh
- String quá dài trong tiếng Việt: test UI với string dài nhất
